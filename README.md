# 📚 BooksBazaar

BooksBazaar is a modern and responsive **Books E-Commerce Web Application** developed as part of the Web-Development-Project repository.  
This project demonstrates front-end web development skills using HTML, CSS, JavaScript, and Bootstrap.

It provides a clean and attractive platform where users can browse books across multiple categories such as Motivational, Fiction, and Spiritual.

---

## 🚀 Features

- 📱 Fully Responsive Design (Mobile Friendly)
- 🎠 Bootstrap Carousel Slider
- 📚 Category-wise Book Display
- 🛒 Add to Cart Button (UI-based)
- ✨ Hover Animations for Book Cards
- 📖 Horizontal Scroll (Amazon-style Book Rows)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  
- Unsplash (for placeholder images)
